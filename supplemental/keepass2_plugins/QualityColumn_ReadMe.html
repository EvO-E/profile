The QualityColumn plugin adds support for a 'Password Quality' column.


Installation

In order to install the plugin, copy the files QualityColumn.plgx and QualityColumn_ReadMe.html into the KeePass application directory (where KeePass.exe is), and restart KeePass.

The 'Password Quality' column can be turned on and off in 'View' -> 'Configure Columns'.

If you wish to uninstall the plugin in the future, simply delete the two files.


Performance

As estimating the quality of a password is a computationally intensive task, the quality column can strongly slow down KeePass.
